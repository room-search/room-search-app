## 주요 기능

### 모든 테마를 한 번에

전국 방탈출 테마를 한 앱에서 검색하세요. 난이도, 공포, 활동성, 스토리, 문제수준, 인테리어 점수는 물론 플레이타임, 가격, 지역까지 한 번에 필터링할 수 있어요. 만족도순, 난이도순, 공포순 등 지금 관심 있는 기준으로 바로 정렬할 수 있어요.

### 딱 맞는 테마 찾기

뭐가 좋을지 모르겠다면 파인더에 맡겨보세요. 두 가지 모드를 준비했어요:

- **위자드** — 5단계 가이드 흐름: 관심 지역 → 공포 취향 → 난이도·활동성 → 스토리·문제 → 시간·예산. 슬라이더 없이 탭 한 번으로 답하는 선택지만 나와요.
- **대시보드** — 모든 필터가 한 화면에 있고, 라이브 매칭 카운터가 조건에 맞는 테마 개수를 실시간으로 보여줘요.

두 모드 모두 매칭 엔진으로 연결돼요. 내 기준에 얼마나 가까운지를 0–100점으로 계산하고, 중요하게 설정한 항목에 가중치를 더 실어 순위를 매겨요.

### 지도에서 바로 찾기

지도 뷰를 열면 전국의 방탈출 카페가 위치별로 표시돼요. 원하는 동네까지 지도를 움직인 뒤 **"이 지역 재검색"** 버튼을 누르면, 현재 보이는 화면 안의 카페만 다시 불러와요. 핀을 탭하면 카페 정보가 뜨고, 하단 카드 레일을 가로로 스와이프해 주변 카페를 둘러볼 수도 있어요.

### 한눈에 보는 점수

모든 테마에는 난이도, 공포, 활동성, 스토리, 문제, 인테리어, 연기력, 만족도까지 8축 레이더 차트가 붙어요. 스토리 중심의 감성 테마인지, 활동량 많은 비명 질주형 테마인지 한 장면에 파악할 수 있어요.

### 즐겨찾기

테마나 카페의 하트를 탭해 바로 저장하세요. 즐겨찾기는 기기 안에만 저장되고, 추가·제거하는 순간 앱 전체에 실시간으로 반영돼요. 새로고침이 따로 필요 없어요.

### 지역 단위로 세밀하게

한 지역만 고르든, 여러 지역을 함께 고르든 자유예요. 서울·경기 같은 시/도 단위로 선택하거나, 강남·홍대·건대 같은 상권 단위까지 좁힐 수 있어요. 선택한 지역은 파인더, 테마 리스트, 지도 어디서든 그대로 유지돼요.

### 취향 힌트 자동 반영

파인더에 "처음이에요"라고 답하면 난이도가 입문자 범위로 조정되고, "가족과 함께"라면 공포와 활동성을 안전한 수준으로 좁혀요. "공포 매니아"라고 하면 가장 무서운 테마 쪽으로 가중치가 기울어요. 내가 준 힌트에 따라 매칭 가중치가 스스로 바뀝니다.

## 시작하기

1. **레포지토리 클론** 후 IDE에서 열기
2. **Flutter 설치** (3.35 이상, Dart SDK 3.9.2+)
3. **의존성 설치** — `flutter pub get`
4. **네이버 지도 API 키** 환경 설정에 추가 (지도 뷰 사용에 필요)
5. **앱 실행** — `flutter run`

별도 계정이나 로그인 없이, 앱을 켜면 바로 테마 리스트에서 시작해요.

## 기술 스택

- **Flutter** 3.35+ / Dart ^3.9.2
- **Riverpod** — 상태 관리
- **go_router** — 탭 중첩 라우팅
- **Dio** — REST 클라이언트 (envelope/auth/rate-limit 인터셉터 포함)
- **Hive CE** — 즐겨찾기 로컬 저장
- **Naver Map** — 지도 뷰
- **flutter_animate** + **staggered_animations** + **skeletonizer** — 애니메이션과 스켈레톤 UI
